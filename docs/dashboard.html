Data Glossary & Explanations

Key Metrics Explained:

  • Key metrics provide insights into detected threats and attack trends.
  • Total Compromised Accounts: Number of detected hacked accounts.
  • Deleted Accounts: Accounts identified as deleted or inactive.
  • Active Malicious URLs: Currently active phishing or attack links.
  • Most Common Attack Method: Predominant technique used by attackers.

Status Indicators:

  • Indicators help classify the threat level of detected URLs.
  • INACTIVE - URL is no longer a threat.
  • ACTIVE - URL is currently a risk.
  • UNKNOWN - URL status is uncertain.
  • UNDER REVIEW - Investigation is ongoing.

Chart Breakdown:

  • Charts provide a visual representation of attack patterns.
  • Attack Methods Distribution: Breakdown of techniques used by attackers.
  • Attacks Over Time: Historical trend analysis of attack frequency.
  • Attack Surfaces: Platforms and applications targeted by attacks.
  • Suspected Regions of Origin: Geographical sources of attacks.
  • Behaviour Types Distribution: Classification of malicious behaviors.

Additional Charts:

  • These charts provide deeper insights into attack characteristics.
  • URL Status Comparison: Distribution of URL threat statuses.
  • Attack Goal Distribution: Objectives behind identified threats.
  • Compromised Account Status: Current state of affected accounts.
  • Method vs Goal Matrix: Relationship between attack methods and objectives.
  • Attack Vectors Analysis: Entry points exploited by attackers.

Detailed Data Table:

  • The table provides comprehensive records of individual cases.
  • Found On: Date when the threat was identified.
  • Username: Affected user's display name.
  • Attack Method: Technique used in the compromise.
  • Target: Platform or service being targeted.
  • Status: Current threat status (Active/Inactive/Unknown).

Dashboard Features:

  • The dashboard offers several interactive features to analyze the data.
  • Search: Find specific entries by username, behavior, or attack method.
  • Filters: Narrow results by attack method or date range.
  • Interactive Charts: Click datapoints to switch views or hover for details.
  • Export Data: Download the filtered results as a CSV file.
  • Detailed View: Click on table actions to see comprehensive case details.
